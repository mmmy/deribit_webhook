🧪 Delta 管理器账户验证测试

✅ 有效账户测试

这些链接应该正常工作：

/delta?account=yqtest /delta?account=main

❌ 无效账户测试

这些链接应该显示错误提示：

/delta?account=nonexistent /delta?account=invalid_user /delta?account=test123 /delta?account= (空账户)

📋 预期行为

  • 有效账户: 正常加载 Delta 管理器页面
  • 无效账户: 显示错误提示 "账户 'xxx' 不存在。可用账户: yqtest, main"
  • URL 清理: 无效账户的 URL 参数会被自动清除
  • 选择框重置: 账户选择框会重置为默认状态

🔧 测试步骤

  1. 点击上面的测试链接
  2. 观察页面行为和错误提示
  3. 检查 URL 是否被正确清理
  4. 检查账户选择框状态
  5. 打开浏览器开发者工具查看控制台日志
